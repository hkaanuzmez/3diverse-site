# 3Diverse – Kurulabilir Next.js 3D E‑Ticaret (TR/ENG)

Bu şablon; Next.js 14 App Router + Prisma + Tailwind + i18n + 3D ürün görüntüleyici + Admin iskeleti içerir.

## Hızlı Başlangıç
```bash
npm install
cp .env.example .env
# DATABASE_URL, ADMIN_EMAIL, ADMIN_PASSWORD değerlerini doldurun
npx prisma migrate dev --name init
npm run dev
```

Admin e‑posta: diverse3d@outlook.com (env ile değişebilir)
